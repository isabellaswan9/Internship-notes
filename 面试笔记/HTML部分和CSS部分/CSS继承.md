## CSS继承属性有哪些

#### 　1、字体系列属性

　　font-family：字体系列

　　font-weight：字体的粗细

　　font-size：字体的大小

　　font-style：字体的风格

#### 　2、文本系列属性

　　text-indent：文本缩进

　　text-align：文本水平对齐

　　line-height：行高

　　word-spacing：单词之间的间距

　　letter-spacing：中文或者字母之间的间距

　　text-transform：控制文本大小写（就是uppercase、lowercase、capitalize这三个）

　　color：文本颜色

#### 　3、元素可见性：

　　visibility：控制元素显示隐藏

#### 　4、列表布局属性：

　　list-style：列表风格，包括list-style-type、list-style-image等

#### 　5、光标属性：

　　cursor：光标显示为何种形态
