#### 正则表达式

##### email的正则表达式

名称部分：[a-zA-Z0-9_-]+

域名部分：[a-zA-Z0-9_-]+(\\.[a-zA-Z0-9__-]+)+

最终格式（名称@域名）：^[a-zA-Z0-9_-]+@[a-zA-Z0-9_-]+(\.[a-zA-Z0-9_-]+)+$

| 量词 | 描述                                  |
| :--- | :------------------------------------ |
| n+   | 匹配任何包含至少一个 *n* 的字符串。   |
| n*   | 匹配任何包含零个或多个 *n* 的字符串。 |
| n?   | 匹配任何包含零个或一个 *n* 的字符串。 |
