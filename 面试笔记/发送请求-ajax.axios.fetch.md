##### ajax中readyState的5个状态及其含义

##### 以下是XMLHttpRequest对象的三个重要属性：

| 属性               | 描述                                                         |
| ------------------ | ------------------------------------------------------------ |
| onreadystatechange | 存储函数，每当readyState属性改变时，就会调用该函数           |
| readyState         | 存有XMLHttpRequest的状态。0：请求未初始化 1：服务器连接已建立 2：请求已接收 3：请求处理中 4：请求已完成，且响应已就绪。 |
